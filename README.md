# Parts Inventory

Browser-based electrical parts inventory backed by Supabase.

## Project files

- `index.html` contains the application markup.
- `style.css` contains the responsive visual design.
- `script.js` contains authentication, inventory workflows, and rendering.
- `supabase/migrations/` contains the required security migration and the feature-extension migration.
- `tests/browser-smoke.html` provides a local browser regression test.

Before deploying the frontend, complete every step in `SECURITY_SETUP.md`.
